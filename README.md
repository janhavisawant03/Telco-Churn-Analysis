# Telco Customer Churn Analysis

## Project Overview
This Power BI project analyzes customer churn patterns in a telecom company.

## Objectives
- Identify customers likely to churn
- Analyze customer demographics
- Study service usage patterns
- Generate business insights

## Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV Dataset

## Key KPIs
- Total Customers
- Churn Rate
- Active Customers
- Revenue Analysis

## Dashboard Preview

![Dashboard](Screenshots/Dashboard1.png)

## Insights
- Customers with month-to-month contracts have higher churn rates.
- Fiber optic users show significant churn.
- Senior citizens have a higher tendency to churn.
